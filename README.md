# MDH-1056 Library System

A library management system for managing books, members, and borrowing records.

## Features

- Book catalog management
- Member registration and management
- Book borrowing and return tracking

## Getting Started

Clone the repository and follow setup instructions to get the project running locally.

```bash
git clone https://github.com/ImJustHenry/MDH-1056-Library-System.git
```
